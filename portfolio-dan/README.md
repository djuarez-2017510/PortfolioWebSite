# 🌐 Portafolio Web - Daniel Juárez

Este proyecto es un **portafolio web interactivo** desarrollado con **React + Vite + Tailwind CSS**, con animaciones fluidas y diseño responsivo. El objetivo es presentar de forma creativa tu perfil como desarrollador y tus proyectos destacados.

# Tecnologías
React
Vite
Tailwind CSS
Framer Motion
tsParticles
AOS (Animate on Scroll)

# Instala dependencias:

-bash
npm install

Ejecuta el servidor de desarrollo:
-bash
npm run dev
